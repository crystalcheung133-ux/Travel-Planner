# Journey Planner v0.1

A standalone pre-trip review portal for Saigon Girls Trip 2026.

## Features

- Landing page
- Trip dashboard
- Participant progress
- Day-by-day itinerary review
- Overall review and final approval
- Admin review summary
- LocalStorage demo saving
- JSON export for review data

## Deployment

Upload the whole folder to GitHub, then deploy with Vercel as a static website.
